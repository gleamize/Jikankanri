# テナント入退室管理アプリ

ブラウザ上で動作する、シンプルなテナント入退室時間管理アプリです。
スマホでの操作に最適化されており、ワンクリックで時間を記録・編集・共有できます。

## 📱 機能

* **簡単記録:** 「入室」「退室」ボタンを押すだけで現在時刻を記録。
* **データ保存:** データはブラウザ（ローカルストレージ）に自動保存され、閉じても消えません。
* **スマホ最適化:** タップしやすいデザインと、片手で操作しやすいレイアウト。
* **並び替え:** テナントの順番をドラッグ＆ドロップで自由に入れ替え可能。
* **編集・修正:** 入力した時刻やテナント名は後からタップして修正可能。
* **データ共有:** 記録したデータをテキスト形式でコピーし、LINEやメールに貼り付け可能。
* **深夜対応:** データ共有時は、深夜（0時〜6時）の記録を「24時〜30時」表記に自動変換して出力します。

## 使い方

1.  **テナント追加:** 上部の入力欄にテナント名を入力し「追加する」ボタンを押します。
2.  **時間の記録:** 各カードの「入室」「退室」ボタンを押すと、現在時刻が反映されます。
3.  **修正:** 時刻や名前が表示されている部分をタップすると、編集画面が開きます。
4.  **削除:** ゴミ箱アイコンを押すと、そのテナントを削除できます。
5.  **共有:** 上部の「データ共有」タブに切り替えると、一覧が表示されます。「テキストをコピー」ボタンでクリップボードにコピーできます。

## インストール不要

HTMLファイル単体で動作するため、サーバーへのアップロードや特別なインストールは不要です。
ブラウザ（Chrome, Safariなど）で開くだけで使用できます。

## 注意事項

* データはご利用のブラウザ内に保存されます。ブラウザのキャッシュ（履歴）を削除するとデータも消える場合がありますのでご注意ください。
